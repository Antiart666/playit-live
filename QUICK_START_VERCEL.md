# ⚡ SNABB START - VERCEL DEPLOYMENT (5 MIN)

## STEG 1: Skapa gratis Vercel-konto
→ Gå till https://vercel.com → "Sign Up" → "Continue with GitHub" → Logga in

## STEG 2: Importera repo
→ Vercel Dashboard → "Add New" → "Project" → Sök "playit-live" → "Import"

## STEG 3: Deploy
→ Acceptera alla defaults → Klicka "Deploy" → Vänta 3-5 min

## STEG 4: Du är LIVE! 🎉
→ Vercel visar din URL: `https://playit-live.vercel.app`

---

## LÄGG TILL NYA LÅTAR (Auto-deploy)

```bash
git add library/
git commit -m "Added new song"
git push origin main
```

→ GitHub Action startar automatiskt  
→ Vercel bygger och deployer  
→ Din app uppdateras inom **2-3 minuter**
